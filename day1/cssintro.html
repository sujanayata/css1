Styling Language – CSS (Cascading Style Sheets) is used to style and format HTML elements, such as colors, fonts, layouts, spacing, and responsiveness

why we need CSS ?

  1. Without CSS → plain website.
  2. With CSS → professional and user-friendly website.
  3. Adds Style – Makes websites look attractive with colors, fonts, and layouts.
  4. Consistency – One CSS file can style multiple pages.
  5. Responsive Design – Helps websites adjust to mobiles, tablets, and desktops.

Three ways to insert CSS into HTML

  1. Inline CSS

    2. Inline CSS means writing CSS directly inside an HTML tag using the style attribute. It is used to apply a unique style to a single element only.

    • Syntax:

      • <tagname style="property:value;">

    • Example:

      • <p style="color":red font-size:18px;">

    • you need to style one element
    • for quick testing and debugging

  • External CSS

    • External CSS means writing all the CSS code in a separate .css file and linking it to the HTML file.
    It is the most preferred method because it keeps structure (HTML) and style (CSS) separate.

    • syntax

      • <link rel="stylesheet" href="style.css">>

      Example:

      style.css

      h1 {
    color: green;
    text-align: center;
        }

      index.html

      <!DOCTYPE html>
    <html>
        <head>
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
            <h1>Welcome to CSS!</h1>
        </body>
        </html>
      Use When:
    • You are working on large or multiple-page websites
    • You want to reuse styles across many pages
    • Best practice for clean code and separation of concerns
